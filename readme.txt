Motivationstipps & Zitate

== Description ==

Dieses Plugin zeigt in Ihrer Sidebar als Widget oder auf Ihrer Seite nach Eingabe des Shortcodes [ds-motivationstipp link="yes"] (ein dezenter Link zum Anbieter wird angezeigt) oder [ds-motivationstipp link="no"] (Link wird nicht angezeigt) automatisch nach jedem Ladevorgang ein motivierendes Zitat an.


== Installation ==

1. Den Ordner 'dirkschmidt-motivationstipp' in das Verzeichnis `/wp-content/plugins/` verschieben
2. Das Plugin über das Adminpanel unter Plugins aktivieren
3. Das Plugin kann über Widgets in die Sidebar eingepflegt werden (Name des Widgets "Motivationstipp") oder durch den Shortcode [ds-motivationstipp link="yes"] in einem Post oder Seite
